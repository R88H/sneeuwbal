# Sneeuwbal Hypotheek Simulator

Een kleine web-app om inzicht te krijgen in het effect van extra aflossen volgens de sneeuwbalmethode op één of meerdere lopende hypotheken.

## Wat kun je doen?

- Meerdere hypotheken toevoegen.
- Per hypotheek kiezen tussen:
  - **Annuïteit**
  - **Lineair**
- Een extra aflosbedrag per maand invullen.
- Vergelijken tussen:
  - **Normaal aflossen**
  - **Sneeuwbalmethode** (extra bedrag naar de kleinste openstaande schuld)

## Starten

Omdat dit een statische app is, kun je `index.html` direct openen in je browser.

Of start een simpele server:

```bash
python3 -m http.server 8000
```

Ga daarna naar:

- http://localhost:8000
